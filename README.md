Veggie Finder locates vegetarian restaurants nearby using the Google Places API!

To deploy locally, create a .env file in the client directory with the contents: 'REACT_APP_API_KEY={your_google_places_API_key}' and another .env file in the server directory with the contents: 'API_KEY={your_google_places_API_key}'. Run 'npm start' in the client directory and 'node index.js' in the server directory. You can then access the application at http://localhost:3000!
